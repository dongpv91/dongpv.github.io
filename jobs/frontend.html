스포카에서 상점을 위한 IT 서비스를 개발할 프론트엔드 프로그래머를 찾고 있습니다.

{% include jobs_spoqa_intro.html %}

업무 내용

  • 웹용 대쉬보드 사용자 인터페이스 개발
  • 웹 기술을 사용한 태블릿 사용자 인터페이스 개발
  • 각 국가별 랜딩페이지 개발
  • 데이터 시각화 프레임워크 개발
  • 스포카 자체 스타일 프레임워크 개발

지원 자격

  • HTML, SCSS, JavaScript를 능숙하게 쓸 수 있어야 합니다.
  • 서비스를 직접 개발하고 운영해본 경험이 있어야 합니다.
  • 반응형 웹 서비스의 개발 경험이 있어야 합니다.
  • Internet Explorer 8 이상의 브라우저 호환성 지원 경험이 있어야 합니다.
  • (우대) 위 기술 스택의 사용 경험이 있으신 분
  • (우대) 일본어와 한국어가 모두 가능하신 분
  • (우대) 글로벌 서비스에 대한 개발 및 운영 경험이 있으신분
  • (우대) SEO를 통한 유의미한 고객 지표 개선 경험이 있으신 분
  • (우대) 오픈소스 프로젝트에 기여 경험이 있으신 분
  • (우대) 단위테스트, 기능테스트, 종단테스트, 빌드 자동화, 지속적 통합 등의 경험이 있으신 분

기술 스택

  • Bootstrap
  • React
  • Python
  • Flask
  • SQLAlchemy
  • AWS

필요 역량

  • 배움에 대한 의욕이 있어야 합니다. 급격히 변화하는 IT 디자인 트렌드 속에서 새로운 물결에 언제든 신속히 대응하고 나아가, 스스로의 역량을 신장하기 위해 언제나 공부를 게을리 하지 않는 분을 구합니다.

  • 깊은 웹 지식이 필요합니다. 많은 부분이 웹 기술에 의존하고 있어 웹 기술 전반에 대한 실무 경험과 최근 기술의 변화에 대한 이해가 필요합니다.

  • 프로젝트의 마지막까지 디테일을 놓지않는 책임감이 필요합니다. 모든 디테일을 섬세하게 챙기고, 완결력 있는 제품을 만들고자 하는 열정이 있어야 합니다.

  • 서비스에 대해 진지하게 고민해야 합니다. ‘정말 중요한 문제를 해결할 수 있는’ 방법을 찾기 위해, 직군이나 분야에 상관 없이 함께 아이디어를 내고 함께 일을 진행할 분을 구합니다.
